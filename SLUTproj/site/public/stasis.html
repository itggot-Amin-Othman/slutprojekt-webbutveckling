Stasis, vad är det och hur används det?

Vad och varför?

Stasis och sass används för att göra det enklare för den som skriver koden samt för den som sedan läser den. Stasis låter dig skriva i flera olika filer som sedan sammanställs till en enda fil. Genom att endast ha en fil kan man effektivisera överföringen av data över internet.

Hur?

Installera först stasis och sass genom att i terminalen köra

gem install stasis
gem install sass

Lägg till filändelsen .erb på alla .html-filer (inklusive .html) och .scss på alla css-filer
Skapa en ruby fil döpt till "controller.rb" i samma mapp som dina html filer och skriv in

ignore( /\/_.*/)
ignore(/.*.sass-cache.*/)
layout('layout.html.erb')

I terminalen, navigera till mappen med dessa filer och kör stasis
Detta skapar en public-mapp med dina genererade statiska sidor.

För att slippa skriva stasis i terminalen efter varje ändring kan du köra "stasis -d 8080" i mappen.
Detta startar en liten server som automatiskt kör stasis vid varje förändring.
