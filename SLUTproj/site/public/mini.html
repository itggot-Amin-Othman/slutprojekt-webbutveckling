(A)minify, vad är det och hur används det?

Vad och varför?

Minify är ett program som tar all din kod och komprimerar den så att man överför mindre data. Den tar alla namn och gör om de till kortare namn som endast den kan förstå.

Hur?

I terminalen kör:

gem install uglifier
gem install cssminify

lägg in denna filen i din site-mapp och till sist skriver du till i din controller fil:

ignore 'minify.rb'

För att sedan köra stasis & minifiera filerna, kör:

stasis && ruby minify.rb
